Books - A Collection

Welcome to the Books collection manager. This is a university coursework assignment, it should be able to perform CRUD operations on a data set containing information on about 10,000 books that was sourced from an account on GitHub who in turn collected it from GoodReads. CRUD operations + some visualisations cover the extent of what was required according to the assignment brief, anything else you see is bonus, because I felt like it :)
